# HR Attrition Analysis

# Project Overview
This project analyzes employee attrition data using Python and machine learning techniques to predict whether employees are likely to leave a company.

The goal of the project was to:
- Identify patterns related to employee turnover
- Build a predictive model
- Evaluate model performance using classification metrics


## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
